Mobile Library

The Mobile Library carries a good range of books, which are available in large print, on tape and on CD. You can request books as you would in any library. The service is free, use your Southampton library card or join on the bus.

The bus features a lift for wheelchairs and anyone who needs help getting up the steps. There is no need to pre-book. Just turn up and staff will help.

The mobile library timetable shows you where you can find the library. Anyone can use the mobile library. It also visits nurseries and pre-schools to offer storytelling and activity sessions to children.

For more information please contact us on 07768 541797
