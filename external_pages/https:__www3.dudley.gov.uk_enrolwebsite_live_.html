Welcome to Enrol: Online Admissions Application

The Enrol application allows you to apply for your childs reception class or, for a child who is transferring to secondary school, to do so via an online interface.

You can create an account, add your child(ren) to the account, and create one or more school preferences for them. Once the preferences have been created, they are available for the Local Authority to view. At the end of the Admissions period, the LA will begin allocating children based on the information supplied in their preferences.

For more information about the admissions process you can look at our website www.dudley.gov.uk/admissions

To begin using Enrol, create an account using the 'Create Account' link which can be found on the upper left side of this page. To return after creating an account, use the 'Log In' link which can also be found on the upper left side of this page. If you are using a tablet/smartphone (or some versions of web browser) access 'Create Account' and 'Login' links using the menu in the top right hand corner.
