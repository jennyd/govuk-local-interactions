Computer Booking System

Welcome to the Libraries Computer Booking System.

If you wish to use a computer in any of the libraries in Torfaen, you can now book a slot in advance to avoid disappointment.

  • You can book up to 8 days in advance
  • You can reserve a computer in any Torfaen Library
  • If you forget what time or library your reservation is for you can check out the find reservations facility

Sessions are initially for a maximum period of 2 hours for adults and 30 minutes for children, but subject to availability this can be extended (adults only).

Book time on a library PC using the online computer booking system.

Please note that you will need a pin in order to access the system. If you do not currently have a pin number, please contact your local library.

A copy of the Acceptable Usage Policy for using Public Access PCs can be downloaded here.
