Planning Applications Search Criteria Description
Case Number (eg 04/00899/FUL) Search for Planning Applications by entering as much information as possible into the fields shown.

Alternatively you may select a particular weekly list by selecting the appropriate date from the drop down list.

You can receive the Weekly Application List by email. Click here to register.
(Please note: We have plans available for Applications back to 1st April 2002 and database records back to 1st August 1991.)

Planning Application documents will be published within 5 days of receipt of a valid planning application. The information is updated daily but is a day behind the present day's date. Comments can be made electronically by using the Comments button available on the application search page or when viewing a particular application.

Further planning information is available via the Planning and Environment section of the website.
