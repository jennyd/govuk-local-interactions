View planning applications

The best way to view current and past planning applications online (from 2008) is to use our online portal.  You can search for applications, appeals and enforcements using keywords, application reference numbers, postcode or single line of address.

This is a new process so we apologies in advance if there are any difficulties accessing the information you require  - please leave us some feedback so we can make sure we fix the issues as quickly as we can.
