Empty homes

We are working with a local housing association to bring empty homes back into use.

Report an empty property

If you’re concerned about an empty property you can:

  • report it online
  • call 0121 704 8000

Can you please include the following information:

  • address of the property
  • type of property (eg. bungalow, flat, semi-detached house)
  • external condition of the property
  • how long it has been empty
  • who owns the property (If known)
  • any nuisance or health hazards associated with the property
  • your contact details, so we can keep you informed while we investigate the property

Can I find out who owns an empty property?

We cannot give out details of empty property owners.

In some instances property ownership can be obtained from the Land Registry.
