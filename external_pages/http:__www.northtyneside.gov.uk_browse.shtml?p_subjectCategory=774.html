Planning applications online

You may view and comment on planning applications online via North Tyneside Council 'Public Access' system.

Please note that full details of correspondence received in relation to individual planning applications are only available to view on the planning case file. These files can be viewed at the Council's Quadrant offices. Please ring the Planning office on 0191 643 2310 beforehand to arrange for the file to be made available.

We are commited to providing the highest quality online planning services. Our Public Access site offers users the ability to:

  • Register and monitor the progress of planning applications.
  • Submit comments about an application.
  • Search a constantly updated list of applications validated and decided each week.
  • View details of planning appeals, planning applications and enforcements.
  • View property details by reference to a map.
  • View Associated Documents, including plans, application forms and decision documents.

If you register as a public access user you can take advantage of additional features which will allow you to track applications and save searches and receive automatic updates.

For more detailed information please see the link to the user guide at the end of this page.

There are also other online services including:

  • Submission of Applications online
  • Submission of Appeals online
  • Interactive Unitary Development Plan
  • Application Forms
  • Committee Reports

Please note that comments made via public access will be published on the Council's website. A copy of your comments can be emailed if you request this. We will also endeavour to correspond with you by e-mail - to notify you of the processing of the application and the outcome of the decision.

If you have any problems using this site or would prefer to view a paper copy of the application they are available to view in the planning reception at the Council officers at Quadrant East between 8:30am & 5pm Monday to Thursday (to 4:30pm Friday).

Any letters of representation should be sent to Planning, 1st Floor Left, Quadrant East, The Silverlink North, Cobalt Business Park, North Tyneside, NE27 0BY

Comments submitted through the website can be viewed within the comments sections of Public Access. Comments submitted using alternative means can be viewed on the application file. Personal information will not be displayed on the web site but letters, including personal information, will be held, and available to view, on the application file.

This site is open to the public and contributors are asked to refrain from making any comments that could be viewed as defamatory or prejudicial as this could result in action being taken against the contributor concerned. Comments made will be screened before publication on the web site and submissions containing such comments will not be displayed.

We will accept comments up to the point at which the application is determined. However it is in your interest to submit comments as early as possible. We will not determine the application before the end of the publicity period. Any parties consulted will have a minimum of 21 days from date of consultation to respond. Details of publicity and consultation can be viewed on the web site.

In the case of a householder application and in the event that the application is refused and subsequently goes to appeal, any representations made about the application will be passed to the Secretary of State and there will be no opportunity to make further representations.

Whilst every effort has been made to ensure the accuracy of data displayed on this site it should not be used as a substitute for carrying out a formal 'Land Charge Search'. Public access provides details of planning applications from 1974 onwards and copies of submitted applications from November 2005.

Please also remember that plans and applications can be frequently amended and if you print off a map there maybe a newer version available at a later date.

At present the system does not allow searches to be carried out using multiple/block addresses or 'land at...' such as 1-3 High Street. Search users will need to identify and search all of the properties / land parcels individually (i.e. 1 High Street, 2 High Street, 3 High Street etc. in order to ensure that the entire history is revealed. If you wish to obtain a full planning history of the site you will need to contact us. There may be a charge for this service.

View applications by Ward

Planning applications which straddle a ward boundary are recorded against the Ward within which the application site primarily falls. If you are searching for applications which straddle or lie close to a Ward boundary you are advised to search separately for records in different wards.
