Report it

You can use this page to report common issues such as graffiti, bins, flytips, litter and weeds.

To start an online report click on one of the links below to load the relevant report form. Please give as much detail as possible as it will help identify what the issue is and resolve it faster. If you leave your contact details you will be given a case reference number which you can use to track your existing reports [external link].

My Darlington+

My Darlington+ is a service which allows you to:

  • Build up a list of links to pages that are of interest to you, and be notified whenever they are updated
