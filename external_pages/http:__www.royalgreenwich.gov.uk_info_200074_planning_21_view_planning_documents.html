View planning applications, decisions and appeals

Where to view planning documents

You can find out about new planning applications and decisions, and all outstanding appeals and recent appeal decisions, using the online planning applications system.

You can also use the online planning system to comment on planning applications.

To view applications, decisions or appeals, use the search tools on the online planning system.

You can search:

  • using keywords
  • using the application number
  • using the first line of the address of the site or property
  • using the postcode of the site or property
  • by the week or month that the application was validated or decided
  • by ward.

Please note

  1. You can comment on applications without registering, but if there is a problem please email the case officer at planningapps@royalgreenwich.gov.uk.
  2. Application registration dates are currently not showing official dates. Whilst this information is amended please contact Planning to confirm specific registration dates, or view the weekly list of registered planning applications.
  3. There remains a separate system to make a planning application. Follow the link on the right of this page to make a planning application on the Planning Portal.

Additional information

The planning applications system holds all documentation submitted by the applicant for each application that is currently in consultation.

Due to limited disk space available, we may occasionally need to remove documents relating to applications that have been determined. We are working to increase the disk space available. In the meantime, if you find a document missing and you would like to view it, please contact our planning team.

Architects' drawings

To view full details of planning applications, including architects' drawings, visit:

  • a local library
  • our planning department's offices.
