Why create an Arena username?

You can log in using your library card number and PIN in the same way as before but with an Arena username you can:

  • Create a wish list

  • Recommend books to friends

  • Rate books

  • Leave reviews

  • Tag books

  • Save searches

  • Create reading lists ﻿﻿
