Planning decision notices search

When a decision is made on a planning application, a decision notice is issued to the applicant or the applicant's agent. The decision notice details key information about the application including the site and the proposal.

If an application has been refused the decision notice will state the reason for refusal, whereas if an application is approved the decision notice will state any conditions that apply to the approval, together with reasons for each of the conditions.

If you know the planning application number you can use the following search facility to access the decision notice. Your search may take a few seconds to complete.
