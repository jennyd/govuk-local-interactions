Public applications search: guidelines for use

The Public Access website has been designed to provide an efficient and easy-to-use tool for you to search for and track applications for planning permission, appeals against refused consent, and planning enforcements.

All public documents relating to an application can be viewed on Public Access and you can you make comments on an application and view the comments made by other people.

You can use Public Access for Planning to view planning applications, appeals and enforcements without creating an account but to take advantage of the advanced features that allow you to save searches, track applications and receive email updates on applications you must create an account.

The following guides will help you make the best use of Public Access:

  • Terms and conditions
  • Creating and managing a user account
  • Creating daily lists
  • Searching for applications
  • Application details and making comments
  • Petitions and commenting on planning applications
  • Deadlines
  • View planning applications
  • Guidance on viewing historical planning application documents

Please read the Terms and Conditions of use before you use Public Access.

Weekly lists

We no longer send out Weekly lists of applications received as this function can now be done in one of two ways using Public Access.

You can visit the Public Access website weekly or monthly and search using the Weekly/Monthly list function as described in the guide 'Searching for Applications'. No user account is required to do this.

Alternatively, you can opt to receive a daily email of the all the applications that have been validated or decided that day. A User Account is required for this. Please read the documents 'Creating and Managing a User Account' and 'Creating Daily Lists' before creating your account.
