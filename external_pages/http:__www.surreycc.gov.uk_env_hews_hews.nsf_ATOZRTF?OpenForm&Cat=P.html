Highways problems and enquiries

You can use this page to report a problem or make an enquiry relating to Surrey's highways.

If you would like to speak to someone or if you would like help completing this form, please call 0300 200 1003.

Please select the subject that relates most closely to your problem or enquiry from the list below. You can use the A to Z to help you find the subject. If we don't deal with this ourselves, we'll direct you to who does.

If you are reporting an emergency or an issue you consider to be dangerous, please do not report it online but instead call our Contact Centre on 0300 200 1003. If you make an online report we will inspect the location within 5 days.
