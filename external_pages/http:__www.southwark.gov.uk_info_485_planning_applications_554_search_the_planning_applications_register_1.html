Search for planning applications

Planning register

The online planning register contains information on all planning applications received by the Council since 2002 and is updated daily. 

You can use the planning register to;

  • view a planning application,
  • comment on a planning application, or
  • find out about a decision that has been made.

Search for planning applications

You can search for an application by its number or address using the quick search page.

You can use the advanced search to search for an application by the date received or decided, address, ward, community council, conservation area, application type (e.g. full or listed building), type of development (e.g. householder, large scale dwellings, or by the name of the applicant or agent).

This register also includes details of planning appeals received and decided.

To find out further information on how to search for a planning application refer to our 'how to' guide.

You can view a interactive map of live plannning notices in consultation period, and comment on applications through this map.

Find out about a planning decision and other planning applications

Use Southwark's interactive map to view planning decisions made in the last 14 days, or to view all planning decisions made since 2005. The map can also be used to view all current and historic planning applications, including those awaiting decisions, and all appeals since 2005.

Submit a planning application

Submit a planning application using the forms and checklists.
