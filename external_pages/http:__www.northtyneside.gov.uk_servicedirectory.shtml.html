This Service Directory includes details of services available to residents through the Council, Partner and/or Voluntary agencies.

There are several ways to search for information on the Service Directory:

  • Use the A-Z to search by category - expanding any sub heading prefixed by a '+' sign to open further details
  • Use the Find a Service search - this will return any service with the word(s) either in the title or description of service (Hint - a common word or phrase, like 'North Tyneside', will produce a large return so you may need to alter your search criteria)
  • Use the Advanced Search - search options include by gender, by age, by area or by key word (e.g. 'Guitar', 'Dance', 'Deaf', etc)
