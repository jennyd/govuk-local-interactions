Welcome to the Leeds online admissions system

You can use this system to apply for a Year 7, Reception or Junior School place to start in September 2016 if you live in Leeds.

If you would like to request a place in any other year group, please call 0113 2224414 or email education.annual.cycle@leeds.gov.uk

Please register for an account using the 'Create Account' button above (you will need to do this even if you've used the online system in previous years).

Once you have created a verified account, you can 'Log In'.
