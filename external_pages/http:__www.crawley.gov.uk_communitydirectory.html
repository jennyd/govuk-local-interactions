Community Directory

The Community Directory contains details of clubs, groups, charities and other organisations who are based in, or support Crawley.  All information in the directory belongs to the groups and although we check entries are eligible this does not imply any endorsement of the contents by Crawley Borough Council. 

If you manage a local group you can submit your details to the directory by clicking on the link below:

  • Add a Group to the Directory

Older People's Directory

The directory includes information on services, support, activities and help available for older people in Crawley. You can find many of these in the Community Directory search. Alternatively, download the Older People's Directory below:

  • Older People's Directory (pdf)
