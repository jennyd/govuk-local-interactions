Planning Applications Enquiries

The database is updated daily at 3am.

By using the links below, you can search the database of Planning Applications, either by using an address or using details of the application.

Using this database you can:

  • Search for planning applications
  • Track the progress of an application
  • View scanned application forms, drawings, decision notices and the conditions/reasons for the decision for applications lodged with the Council.
  • You can also search this database for appeals lodged and decided
  • Comment electronically on applications and view officer reports

View guidance about how to make representations on a planning application and what happens after representations have been made

To begin your search, please tick the box to confirm you have read the Council's disclaimer.
