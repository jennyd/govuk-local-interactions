Search Planning Applications

To search and view the planning applications, please go to our PLANNING APPLICATION SEARCH PAGE. 

 

A SPATIAL SEARCH facility (opens in a new window) is available as an alternative means of searching and provides an interactive map of planning applications registered for each year from 2001 to date. 

 

To use the spatial search, simply click on 'map legend' and select the year(s) that you want to view.  You can find a specific site by entering an address in the address field or by dragging the map around and using the zoom buttons.  Click on a specific planning application (denoted by a red line on the map) to view information about it including application type, address, proposal and decision. 

 

The plans can be viewed by simply clicking on the 'more info' hyperlink which appears below the basic information.

 

 

Query Builder

Query Builder can be used to view the extent of the area relating to a Planning Application. By clicking on the location of the Planning Application, the area will be highlighted on the map.

To use Query Builder, you will need to select the following options;

-      Select application from the drop-down list.

-      Click ‘Go’

-      Click on the ‘Location’ in the results to go to the relevant location.

 

-      To do another search, click on the ‘x’ next to the Planning Year (e.g. RCC Planning 2013)

-      Make sure to reselect ‘All Data’ from the Query In field before doing your next search.

Plans can be viewed by clicking the ‘more info’ hyperlink, however, please be aware that only limited information is available prior to April 2013.

 

Important: please read the disclaimer at the end of this page before using the above search facilities.

 

Viewing the plans and documents

Plans and other images are uploaded in Adobe PDF format. To view these you will need to have Adobe Acrobat Reader installed on your computer. This can be downloaded free of charge from Adobe's website (external link - opens in a new window).

Some files are also uploaded in 'JPEG' format. Most operating systems include software for viewing these, for example, 'Paint', 'Windows Picture & Fax Viewer', 'MS Office Picture Manager' etc. More details can be found on the Fileinfo website (external link - opens in a new window).

Please note: This service may only be available between the hours of 0645 and 2145 (6:45am & 9:45pm) due to an essential daily backup of the planning database which takes place overnight to minimise inconvenience.

Copyright Notice

Plans, drawings and material submitted to the Council are protected by the Copyright Acts (section 47, 1988 Act). You may only use material which is downloaded and/or printed for consultation purposes, to compare current applications with previous schemes and to check whether developments have been completed in accordance with approved plans. Further copies must not be made without the prior permission of the copyright owner.

For applications which have been determined, the council is unable to supply copies of any plans/drawings without the express written consent of the originator.
