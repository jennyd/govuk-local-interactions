Planning Application and Appeals - Quick Search Page

These Planning Application Search Pages enable you to search for planning applications that have been registered and/or determined by the Havant Borough Council. The data that is displayed on these pages is updated daily.

  • If you know the application number (you will find this number on the correspondence that you have received from us) you can use the 'Application Quick Search'.
  • To search for a street name, town or postcode use 'Search by Address'.
  • The Council publishes a weekly list of all applications registered as valid during the week in each Ward. To view one of these lists use the 'Weekly List Search.
  • For further search options, for example to search for an application type or applicant's name, use the 'More Search Options'.

Information about planning and the ability to submit an on-line application is also available through the Planning Portal. If you have any further enquiries then please call us on 023 9244 6015. The line is open from 9am - 5.30pm Monday - Thursday and 9am - 5pm Friday, or contact planning.development@havant.gov.uk for further information.
