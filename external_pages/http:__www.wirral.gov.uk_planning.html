Planning Application and Appeals - Quick Search Page

These Planning Application Search Pages enable you to search for planning applications that have been registered and determined by the Wirral Metropolitan Borough Council. The data that is displayed on these pages is updated daily.

  • To search for a street name, town or postcode use 'Search by Address'.
  • For further search options, for example to search for an application type or applicant's name, use the 'More Search Options'.
  • The Council publishes a weekly list of all applications received in each Ward. To view one of these lists please select the Ward and a date from the 'Weekly List Search'.

Information about planning and the ability to submit an on-line application is also available through the Planning Portal. If you have any further enquiries then please call us on 0151 606 2020. The line is open from Monday to Friday 9am to 5pm, or contact planningapplications@wirral.gov.uk for further information.
